# Sistema CRUD de Productos y Clientes

## Descripción del Emprendimiento

Este proyecto es una aplicación web desarrollada para la gestión de productos y clientes de una tienda.  
El sistema permite registrar, visualizar, editar y eliminar productos utilizando una interfaz web conectada a una API desarrollada con FastAPI.

El objetivo del sistema es ayudar a pequeños negocios o emprendimientos a organizar su inventario y la información de sus clientes de forma sencilla y digital.

---

## Problema que Resuelve el Sistema

Muchos pequeños negocios manejan su inventario de forma manual o en cuadernos, lo que puede generar errores, pérdida de información o desorganización.

Este sistema permite:

- Registrar productos con código, descripción y precio
- Visualizar todos los productos en una tabla
- Editar información de productos
- Eliminar productos
- Gestionar información básica de clientes

De esta manera se mejora la organización y el control de la información del negocio.
